# 主角设定

## 男主：林烬

身份：青云城林家少主，曾是天才，十三岁开六脉，却在十五岁那年被人废掉经脉。
性格：隐忍、果断、重情，但不圣母。
金手指：一柄锈迹斑斑的断剑，内有上古剑灵“玄衣”。
隐藏身份：他不是废脉，而是体内有“天生剑骨”，被人强行封印。

核心成长路线：

从废脉少年 → 断剑传人 → 剑骨觉醒 → 斩宗门天骄 → 查清灭族真相 → 破天门。

## 女主：沈青凰

身份：大玄皇朝长公主，表面冷傲，实际背负皇族诅咒。
修为：前期远强于主角。
关系线：一开始只是利用主角手中的断剑，后来发现主角是唯一能斩断皇族诅咒的人。
人设：高冷、聪明、有手段，不拖后腿。

感情线不要太慢，可以走“互相试探 → 生死合作 → 彼此信任”。

## 反派设定

前期反派：萧寒

主角未婚妻的师兄，玄阳宗天才。
特点：表面正派，实则当年废掉主角经脉的人之一。
作用：负责前期打脸和制造压迫感。

中期反派：林家大长老林啸

主角族中叛徒。
当年为了夺取家主之位，勾结外敌，害死主角父亲。
作用：推动主角回归家族复仇。

终极反派：天门使者

来自九州之外，三百年前斩断龙脉的人。
真正目的：圈养九州武者，把九州当成“养剑之地”。
